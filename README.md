# 🌌 NovaStudio - Creative Digital Excellence

NovaStudio is a premium, high-performance landing page for a creative agency, built with a **Python (Flask)** backend and a stunning, modern frontend. It features a sophisticated design system emphasizing glassmorphism, vibrant gradients, and immersive animations.

![NovaStudio Preview](static/img/hero_visual.webp)

## ✨ Key Features
- **Modern UI/UX**: A state-of-the-art dark theme with curated HSL color palettes and glassmorphic components.
- **Dynamic Hero Section**: Interactive background with AI-generated 3D visuals.
- **Performance Optimized**: Lightweight vanilla CSS and JS for blazing-fast load times.
- **Scroll Reveal Animations**: Elegant entry animations for sections using Intersection Observer API.
- **AJAX Contact Form**: Fully functional contact form with real-time feedback and server-side integration.
- **Responsive Design**: Seamlessly adapts to mobile, tablet, and desktop screens.

## 🛠️ Technology Stack
- **Backend**: [Python 3.11+](https://python.org) + [Flask](https://flask.palletsprojects.com/)
- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), Modern JavaScript (ES6+)
- **Icons**: Font Awesome 6.4
- **Fonts**: Google Fonts (Outfit & Inter)

## 🚀 Getting Started

### Prerequisites
- Python 3.11 or higher installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Apurbo77/hudai.git
   cd hudai
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
Run the Flask development server:
```bash
python app.py
```

Open your browser and navigate to `http://127.0.0.1:5000` to view the site.
